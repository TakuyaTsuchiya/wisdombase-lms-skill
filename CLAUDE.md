# WisdomBase LMS Skill

## スクリプト生成ルール

- スクリプトファイルは `~/Desktop/wisdombase-scripts/` に出力する
- 生成後は `open -t` でテキストエディタで開く（`open` だけだと .js はブラウザで開く）
- ExamLecture のタイトルは **35字以内** にすること（超えるとサイレント失敗する）
- クイズ・課題は **非公開（下書き）** で作成し、公開はユーザーの指示を待つ

## ExamLecture 作成の注意

- `X-Requested-With: 'XMLHttpRequest'` を **付けない**（付けるとリダイレクト先が変わりIDを取得できない）
- レクチャーIDは **リダイレクトURL** (`res.url`) から取得する（ページ再取得で最後のIDを拾う方式は、既存レクチャーがあると誤ったIDを返す）

## コース構築の推奨フロー

1. Step 1: 動画レクチャー作成
2. Step 4: クイズ作成（**非公開**、順番変更あり）
3. Step 5: 実習課題作成（**非公開**）
4. Step 6: トラック制限設定
5. 順番確認・必要に応じて並び替え
6. 公開はユーザー指示で一括実行
